# 安装参考

1，安装参考地址：https://my.oschina.net/edwardge/blog/1632206?tdsourcetag=s_pctim_aiomsg 社区版下载地址 (opens new window)

# 1，解压安装包
tar -zxvf neo4j-community-3.4.10-unix.tar.gz
# 2，修改配置文件为远程可访问，查看
cd neo4j/conf
vim neo4j.conf 
	# dbms.connectors.default_listen_address=0.0.0.0	# 将此处前面的注释符去掉
./neo4j console
# 3，如果一切正常的情况，开浏览器，输入url：http://172.30.154.245:7474/browser/
# 默认用户名密码  neo4j/neo4j	neo4j/123456
