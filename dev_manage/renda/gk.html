# 战术策略方面

三三段式布局

每个课程的连续考试题如名词1、2、3……30，按照A、B、C划分为三个部分，每部分包含1~10道题，对每个段进行背诵复习，方便记忆。如果困难，可以依照此方法对某小段再分。

  1. 历年国考题：以其为基础。
  2. 选专业：选择自己想考的专业。我的选择是信息资源管理+情报学。选择后，国考的论述题就只背这两个专业的，一定要多看多背。
  3. 三三段式：对名词、简答、论述，按此方法划分。
  4. 背诵技巧：信息资源管理、图书、情报、档案的名词、简答题全背；论述题只背选择专业部分（不要背其他科的，否则任务量会翻倍）。摘抄、背诵、思考。
