# 情报学科笔记

培训机构：唯学教育网络学习平台
培训老师：北大-赵丹群教授
PPT

# 情报学基础

情报：很早之前Information之前翻译成情报，现在翻译成信息。现在有时叫做Intelligence(智慧)，竞争情报信息简称CI(Competitive Intelligence)

信息：
狭义： 消除X种不确定性东西；
广义： 泛指一切事物。

语法、语义、语用（套话|普世）

情报最早发展： 军事情报（1946年二战后兴起） ——> 各行各业（科技、商业等领域）

情报是信息的子集

地球上不停运动产生的事件出现事实
信息链： 事实【上游部分】 ——> 数据(有用) ——> 信息 ——> 知识(利用) ——> 智能【下游部分】

好多科都会出现三种写法： 定义、类型(特点)、功能(构成) 【套路】

信息工作 ——> 信息机构

类型 ——> 标准（两者意义相同）

体系结构： 数据处理系统、管理信息系统、决策支撑系统（低级 ——> 高级）
其他结构： 专家系统、办公自动化系统

网络系统结构： 集中（80年代） ——> 分布（现在【C/S、B/S】）

微观 <——> 中观 <——> 宏观 【套路】

商品(微观) <——> 市场运行机制(中观) <——> 国家地区产业(宏观) 【套路应用】

# 信息计量学基础

套话： XX学科是研究社会XX现象及其本质和规律的一门学科。

文献分散定律： 布拉德福定律（2080规律【20%、80%】）
论文作者分布定律： 洛特卡定律（倒数平方规律 【f(x)=C/x2】）

20世纪，对社会贡献最大的学科，计量学科
咱书上只列举了前三种，实际上是有五种。
文献计量学、信息计量学、网络计量学、科学计量学、知识计量学

产品： 自给自足东西
商品： 市场需求产生的（价格受供需关系影响）

价值和价格是不同的东西

理论 ——> 技术 ——> 应用 【套路】

产品和商品都会有一套理论、结合XX技术、产生了XX应用（解决XX问题）

# 信息检索

赵老师是情报学专业的
数据类型： 文献数据、事实数据等
三分（四分）：文本检索、数值检索、音视频检索（音频、视频）

CBR（Content-Based Retrival）: 基于内容检索

老师的课件90%的书上知识都涉及到

情报学1946年问世，先应用在科学计算上，后发展到信息检索

搜索依赖症

辅助性工具： 相关性排序、PageRank、文档价值、竞价排名（百度）

  2. XX技术手段日益现代化。

  2. 信息搜索

  4. 评估

# 信息服务

马斯洛需求
真实需求 ——> 感知需求 ——> 表达需求 ——> 满足需求
（上述过程，在实际转换过程中存在损耗）

信息产生与分布中的马太效应
“凡是多的，还要给他，叫他有余；凡是少的，就连他所有的也要夺过来。”人类社会特有的选择机制（社会惯性），“强者愈强，弱者愈弱。”
