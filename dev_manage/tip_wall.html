# 头脑风暴

星云为什么是扁平的？

原因是星体自转影响，每个星体围绕中心旋转，只有在这个力范围内才会被束缚，否则该星体就会流失，旋转的方向时无规则的， 一旦确定下来后，就会稳定不变，直到有外力改变这种状态为止。

人为什么会思考？

人是有一堆动态物质构成内部环境，这种内部环境与外界环境相互作用，相互影响。在环境作用过程会不断形成新的反射弧， 堆积起来后就是我们所说知识、经验、记忆。这些东西有什么功能呢？逻辑计算，当面临新情况时，通过计算得出结论，从而 影响行为。所以说一个人的成长，外界环境很重要。

继承的价值？

人体在进化的过程中，从祖先那里不断继承下来很多东西，例如生理上（DNA、血液，性等）、心理上的。有很多先天就具备的功能， 所以让机器转变“人”，只要不断迭代进化，结果肯定是毋庸置疑的。

生命的诞生及人类思考物理过程

生命从无机世界到生物世界诞生过程。 思想来源于生物电，这种电能在不停的变化，导致人的动作也在变化，反映在人身上就是思想行为的变化
