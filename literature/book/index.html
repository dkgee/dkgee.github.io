# 书籍

记录觉得非常不错的书籍

生活中的称赞语

  1. 亲爱的，你做的非常棒。但是如果将xx高一点，会更好。

  2. 非常感谢，您的帮助。xxxx

  3. 美女/帅哥，能帮下忙吗？

  4. 你的xx真漂亮

  5. xx，这件事做的非常棒。

  6. 谢谢，我们会再接再厉。

  8. 哪里，你比我多读了几本书。

记录原则

  1. 需要反复阅读；
  2. 引人深入思考，值得分享；
  3. 对日后有影响，具有参考价值。

# 学科分类

通常都是研究XX社会现象及规律，从而发展形成一门学问。对事物的分析方式如下：

定义（概念属性）、类型、结构、功能；
微观、中观、宏观；
理论、技术、应用。

如果研究某一门学科，就要结合上面规律对其进行研究分析，归纳总结。

  • 学科层级
    • 第一类：马列主义、毛泽东思想等（最顶层学科）
    • 第二类：哲学 （顶层学科，如自然哲学、社会哲学）
    • 第三类：
      • 社会科学（例如政治、法律、军事、经济、文化、教育、语言、文字、文学、艺术、历史、地理等）
      • 自然科学（例如数学、物理、化学、天文学、生物科学、医药、卫生、农业、工业、交通运输、航天等）
      • 综合性图书（档案学、情报学等）
