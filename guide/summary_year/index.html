# 总结与规划

回顾这一生，每天在瞎忙些什么，得有目标的去生活。

历年回顾

【2021年度总结】 【2020年度总结】 【2019年度总结】 【2018年度总结】 【2017年度总结】

2023规划

  1. 找一门通过资本自动长久赚钱门道；
  2. 考博士？；

2022规划

  1. 完成人大在职研究生学业，获取人大硕士学位；
  2. 通过信息系统项目管理师软考；
  3. 最好能还完房贷和借贷；

2021规划

  1. 完成小论文、大论文，最好能及时通过人大论文答辩；
  2. 工资能够有所提升；
  3. 每月进行一次团队管理、项目管理、技术管理三方面总结与反思；
  4. 考取PMP资格证；
  5. 将体重控制在78kg左右，养成跑步拉伸习惯；
  6. 深入学习软件开发相关技术(NLP方向、Kali系统使用)，完善dkgee文档网站；

2020规划

  1. 完成人大在职研国考；【已达标】
  2. 工资能够有提升；【已达标】
  3. 完成自身管理能力的提升（说话水平、项目管理）；【已达标，有待后续进一步提升】
  4. 通过健身将体重控制在78kg左右，养成跑步拉伸习惯；【偏离太大，现在90.3kg，需严格控制饮食、锻炼、作息】
  5. 深入探讨软件开发相关技术，将dkgee文档网站打造的更加全面，便于知识理解运用。【已达标，后续保持提升】

4月中旬计划：做好回郑州准备，7月离职，把东西都转到郑州去，国考请假来北京考试，不影响。放开手臂去干，不破不立。

未来5年目标：还完房贷，转变事业发展方向，逐渐转向一个小资本商角色，为社会提供一份简单的服务，来讨活未来。

2020-11-03 总结
目前来看上述五个目标，有80%已经打标了，但是也还有很大的提升空间，特别是体重控制方面，年初增长到84kg，到目前 的90.3kg，体重有了显著的增长，这是因为日常缺乏锻炼导致，后续需要控制饮食，加强身体锻炼。

中午只吃一碗米饭，晚饭吃过后，要走一万步，每晚睡前开合跳三组，早上6点起床，去公园跑步1小时。

  2. ZZ房子装修完成；
  3. 技术提升到更规范稳定的水平：项目快速构建、团队建设；【半拉子】
  4. 上下半年各出去旅游一次；【未去做】
  5. 思考如何经营一家公司；【未思考】

  1. --人大在职研校内题库考、国考；
  2. --工资能够突破15k+；
  3. ==CTFX项目交接，换个新工作环境；
  4. 技术提升到更规范稳定的水平：深入理解应用微服务；
  5. 提高管理水平：团队建设、人员沟通分享、视野规划；
  6. 学习Python。
