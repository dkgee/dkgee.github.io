# 总结与规划

回顾这一生，每天在瞎忙些什么，得有目标的去生活。

马云：人的一生建议

  1. 20岁到30岁的时候，跟一个好老板，加入一个好单位，学点东西很重要；
  2. 30岁到40岁的时候，要做自己最想做的事情；
  3. 40岁到50岁的时候，一定要做自己最擅长的事情，别40岁到50岁还傻乎乎去转个行（可能会成功，失败概率太大）；
  4. 50岁到60岁的时候，把所有的精力培养年轻人；
  5. 60岁以后花在自己孙子身上就行了，回到家里多呆呆；
  6. 70岁以后晒晒太阳，沙滩上走走，有机会的话没机会村里转转也挺好.
