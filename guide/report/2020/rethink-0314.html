# 自我反思

金无足赤，人无完人，每个人都有优点和缺点。我不能做到足够完美，但是我可以通过日常的反思，来进行自我提高和改进。 优点和缺点不一定是一成不变的，随着时间的累计，优点可能也会变成缺点。缺点也可能会变成优点；优点在随日月的增长 过程中，日渐平庸，从而不在明显。缺点则是，在随后日月里，没有提高，依然是缺点。所以一个人要坚持自我反思，自我 改进，自我提高，否则，终究会被无情的岁月淘汰掉。

# 当前优缺点【动态更新】【自我观察】

优点

  1. 定期会进行思考总结。
  2. 喜好书法，偶尔会练习。
  3. 性格沉稳，遇事稳重。
  4. 有收藏整理学习知识的习惯。
  5. 最近一周有吃过晚饭出去散步。
  6. 最近有读新书，并坚持在读。
  7. 最近一周有跟家人保持沟通和联系。

缺点

  1. 克服不了内心的欲望【好色】，多次训诫均无效。💩【长久下去对身体不好】
  2. 日常工作做事爱好拖沓。不到逼不得已得情况，才会去做。【行动力太弱，效率太差】
  3. 学习办公时，总是弓着腰。【容易驼背】
  4. 自律能力太弱。【不能成大事，任何一个能成大事的人，首先就要学会管住自己，与不良的欲望做斗争。】
  5. 不善表达，爱好显摆。【学着明白事理，用心想着沟通表达，切勿盲目自信】
  6. 最近一周没进行Keep锻炼。【改进建议，每周进行两次锻炼】
  7. 没有按时【每晚11点】睡觉，最近熬夜玩吃鸡游戏到半夜一两点。【戒掉游戏，并向外界宣布】
  8. 对待自己不喜欢的事或人，有歧视心里。【要改进，要懂得并尊重别人】
  9. 在手机上，实事新闻看的太多。【浪费时间，改进建议，每天看中午晚上的新闻30分即可】
  10. 最近两个月体重增加过快。【要控制饮食】
  11. 肝不好。【戒酒戒烟，戒腥辣刺激食品】
  12. 左脚踝疼。【不要穿着拖鞋乱蹬墙角，适当揉揉】
  13. 最近一周国考题没背多少。【因为背诵疲劳，分阶段背诵】
  14. 沉迷在自我的小圈子中，没有走出去。【要多和朋友们沟通，结交新的朋友】
