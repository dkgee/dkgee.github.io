# 日常总结

2020-04-02 XX会议经验总结
  1. 【大BOSS讲话思路】全国大局观 ——> 企业管理 ——> 小团队 ——> 员工个人；
  2. 【大领导总结思路】概括中心思想、核心内容再次阐述；

这次会议我认为主要是为了安抚人心。公司管理制度改变必将引起很大体系变动。这次变动公司领导层也有了明显了变化。虽然对于这次奖金制度 方面有些不满，但是，也折射出大BOSS真实的想法，就是要把一部分人干掉。旧人换新人，必然带来改变，即使这改变是有风险的。有人的地方就 有江湖，即使我换一个地方，也还会这样，这就是社会规则。

尊重和拍马屁有什么区别？尊重是在环境相互平等的情况下，才可能会有尊重。拍马屁则是顺毛捋，尊重不会带来多大利益，但拍马屁则可能会。

2020-03-09 会议经验总结
  1. 【同事经验】给领导汇报时，按项目分类列别阐述；跟领导对话时，用“您”，不要用“你”；
  2. 【同事经验】人员职责要分配清楚，如产品经理、研发负责人、研发管理、运营管理；
  3. 【领导经验】关心人的问题，如团队成员的人生安全；
  4. 【领导经验】关心客户的问题，怎样与客户沟通的，客户的关注点是什么；
  5. 【领导经验】关心产品问题，我们的产品是长久迭代做出来的，产品负责人要学会分享经验；
  6. 【领导经验】对自己行业的定位比我要清晰，咱从事的是信息安全监管行业。
