# 生命过客

  此刻是清晨五点五十三，睡醒之后就不想再睡了。脑海总是回忆昨晚与曾老师的对话，唉……世间无所谓高低贵贱，世间无所谓 侧眼旁观，世间无所谓海誓山盟。为了更好的生活，大家各奔前程。
  曾老师和我是高中同学，要好的兄弟哥们。高考后，我们一起报的的都是郑州大学，到校报到后，才发现郑大真的很大，他们 专业在郑大南校区，我们则是在北校区，两个校区都在市区，不算太远，但也不算太近，坐公交半个小时左右。当学生那会儿穷，除了家里给 了点生活费，就只能自己想办法了。跟同学一起打零工，挣点外快，那时的日子对于现在的我来说，我应该感谢！让我在工作、学习、朋友、 生活间协调，期间身边遇到过各种事情，包括坑蒙拐骗、青年早逝等在电影里才能发生的事情。真的挺好。整个人在大学环境熏陶后，成熟了好多。不再 是单独的依赖父母，依赖老师。想想那时的自己，还是挺努力学习的，还拿过两次奖学金，请几个好朋友一起吃饭庆祝。都是高中同学，老家那 里一块。其中有两位女生，两位男生，本想追人家的，结果都没有成。后来就不了了之，大学毕业后，就各奔东西，之后的日子渐渐就失去了 联系。但男生还好，我们一直保持着联系，过年还会去人家里走走玩玩，大学毕业头两年是最开心的时候，工作让自己饿不死的情况下，稍微有 点零花钱就到处逛，也是在那个时候，我去过成都、武汉、南京、上海，去上海在曾老师那里凑合着一晚上。唉……当时自己真穷！但过的很开心， 也是在那段日子里，认识我现在的媳妇，后来我又去了广东和深圳。特别在广东出差那会儿，我媳妇跑去了，我们一起在广东那边逛着玩了好几天， 当时的共享单车超级多，我们骑着单车在广东的风景地、大学校园、商业街一路跑下去，令人难忘。哎呀……跑题了，今天主要还是纪念下曾老师的 爱情事业发展的。当年，曾老师谈了一个，我们聊天时，我建议他就专心致志，后来他好像也就是专心致志。跟他们旁边校区的那位妹子谈了好久 好久，算下来有七八年了…… 太可怕了！谈到这个时候，如果不结婚，那就没啥盼头了，成不了亲人就成陌生人。就在前短时间，曾老师断然挥手 跟人家说拜拜了。是啊，再不拜拜花儿都卸了。嗯……江山如此多娇，引无数英雄竟折腰，俱往矣，数风流人物，还看今朝。毛老的这首诗还是那么 意气风发，令人性情豪迈！
  昨晚和曾老师视频聊了老半天，大致情况也了解了一下。新找妹子，非常nice，我只能说曾老师做的对！赶紧吧！赶紧把婚结喽！ 喝喜酒去！新妹子还是上海本地人，两人是同事，在一块办事也好几年了，妹子对曾老师展开激烈的追求，曾老师在秀色可餐和男人仅有的一点 抵抗之力下还是缴械投降。昨晚看过妹子照片，眉清目秀，笑逐颜开，是个好妹子，作为兄弟，我为之感到庆幸和祝福。都三十好几岁的人了，同 龄人的娃娃都会打酱油了，抓紧吧。
  昨天下午还是公司的年会，由于疫情，今年的年会比较特别，是在线上举办的，活动也异常热烈。在整个过程中，我也算全程参与， 公司给我们放了半天假，在中间环节，很幸运中了个一等奖"小米青春版投影仪"，京东查了下，也两千多块大洋呢！唉……来RZX已经六年有余， 这一路走来，经历了大大小小十几个项目，旁边的同事换了一茬又一茬，熬走了那些老同事，自己也成老员工了，昨天年会的五年忠诚贡献奖，自己 也是榜上有名，太老了，过程只有自己知道！年终奖胡老大前几天跟我透过气，跟前年差不多，是啊！去年业绩不好，你还想咋滴，有的给就不错了， 即使自己换了一家，也不一定会有这么好，还是好好学习，提升自己，升华自己吧！把过往当作历史，迈上新的里程，向着更宽广的道路前进。
  昨晚和曾老师视频时，说我胖了，得减肥呀！这几天忙着学PMP课程，把锻炼身体的事情又仍一边了，还是得跑步呀，不然解决不了 根本问题，即使每天自己心里安慰出门遛个十里路，也还是解决不了根本性问题。冬天冷，不能跑，这是脑海里给自己的灌输的概念，是真的冷吗？ 还是自己压根就不想锻炼呢？如果不是因为风大、下雪等硬性要素，该跑还是得出去跑，晚饭少吃，甚至不吃，早上烤面包，跑起来吧！还记得当时 在郑州七天酒店电梯门看到的那句话吗？人生，因选择而璀璨，也因选择而暗淡，无论年龄几何，只要选择年轻，你的人生就注定与众不同。
