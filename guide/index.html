# 缩进与换行

  1. 缩进：输入下面的代码，然后紧跟着输入文本即可。注意分号。
半角的空格：  &ensp; 或 &#8194;
全角的空格：  &emsp; 或 &#8195;//首行缩紧，使用这种，一个空格等于一个汉字宽度
不断行的空格：&nbsp; 或 &#160; //同一行内，可以通过这种方式在字体间加空格
